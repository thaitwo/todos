## Todos App

> A task management application to help users accomplish daily tasks and be more productive

Visit app: [https://thaitwo.github.io/todos/](https://thaitwo.github.io/todos/)

### Technologies Used

- [jQuery](https://jquery.com/)
- [Webpack](https://webpack.js.org/)
- [SortableJS](https://github.com/SortableJS/Sortable)
- [Sass](https://sass-lang.com/)
- [store](https://github.com/nbubna/store) (localStorage)
- [uuid](https://github.com/kelektiv/node-uuid)

### Local Environment

**Run App**

To run the app locally, go to the root folder in the command line and run:

```
npm start
```

**Compile Code**

To compile the changes made in development, go to the root directory for this repository in the command line and run:

```
webpack --watch
```
